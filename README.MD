# 📡 Bluessager (Mesh Messenger)

**Bluessager** — это децентрализованный кроссплатформенный мессенджер, работающий без серверов, интернета и мобильной связи. Приложение использует технологии Bluetooth Low Energy (BLE) и Wi-Fi Direct для создания локальной самоорганизующейся сети (Mesh-сети) между устройствами пользователей.

В условиях жесткой цензуры, блокировок провайдеров (ТСПУ) или полного блэкаута сети, Bluessager позволяет людям оставаться на связи, обмениваться сообщениями и тяжелыми файлами напрямую — от устройства к устройству (Peer-to-Peer).

---

## 🚀 Как это работает?

Вместо того чтобы отправлять ваши сообщения на удаленный сервер в другой стране, Bluessager использует ваш смартфон как самостоятельный маршрутизатор (ноду).
1. **Радар (Discovery):** Приложение постоянно сканирует радиоэфир вокруг вас в поисках других пользователей Bluessager.
2. **Связь (P2P Cluster):** Когда пользователи найдены, устройства автоматически "сцепляются" друг с другом по Bluetooth или Wi-Fi, формируя единую защищенную сеть.
3. **Дальнобойность (Mesh):** Если вы хотите отправить сообщение другу, который находится в 1 км от вас, но между вами есть другие пользователи Bluessager — ваше сообщение будет передано "по цепочке" от телефона к телефону, пока не достигнет адресата.

## ✨ Текущие возможности (v1.0)

- 📡 **Автономный P2P-чат:** Отправка текстовых сообщений напрямую между смартфонами без Wi-Fi роутера и сотовой вышки.
- 🆔 **Уникальные идентификаторы:** Генерация личного UID-адреса для каждого профиля. Добавляйте друзей и создавайте группы, не делясь номером телефона.
- 👤 **Анонимные профили:** Работа без номеров телефона, почты и регистрации. Вся ваша "личность" — это криптографический UID.
- ⚙️ **Гибкая маршрутизация:** Выделенные вкладки для локальных радаров, сохраненных друзей и групповых чатов. 
- 🤖 **Поддержка Android 10-14:** Корректная работа со строгими разрешениями Bluetooth и геолокации в новых версиях ОС.

## 🔮 Планы на будущее (Roadmap)

Мы находимся в активной стадии разработки. Вот что появится в ближайших версиях:

### 1. Глубокая Mesh-маршрутизация (Store-and-Forward)
Реализация протокола "Сплетни" (Gossip) для связи на огромных расстояниях. Сообщение будет сохраняться в зашифрованном виде на телефонах случайных прохожих-пользователей Bluessager и передаваться дальше, пока не встретит адресата (даже если получатель за много километров от вас).

### 2. Обмен тяжелыми файлами (Wi-Fi Direct)
Бесшовное переключение с энергоэффективного Bluetooth (BLE) на скоростной Wi-Fi Direct для передачи фото, видео и документов размером до 1 ГБ на скорости до 250 Мбит/с.

### 3. Пробитие "Белых списков" ТСПУ (Gateway Mode)
Если хотя бы один телефон в Mesh-сети имеет доступ к интернету, он сможет работать как шлюз (Gateway) для всех остальных устройств в радиусе действия. Дополнительно будет реализована маскировка пакетов (на базе алгоритмов `zapret`) для заворачивания запрещенного трафика в разрешенные "белые" HTTPS-домены.

### 4. Кроссплатформенность (iOS)
Интеграция с Apple `MultipeerConnectivity` для разрушения барьера между экосистемами. iPhone и Android смогут находить друг друга по BLE и создавать локальные хотспоты для обмена данными.

## 🛠 Установка (Для пользователей)

Так как проект пока находится на стадии активной разработки, готовые `.apk` файлы публикуются в разделе **[Releases]** (или скачиваются из раздела **Actions**).
1. Скачайте файл `app-release.apk` на свой Android-смартфон.
2. Откройте файл и разрешите "Установку из неизвестных источников".
3. **ВАЖНО:** При первом запуске дайте приложению **все** запрашиваемые разрешения (Локация, Устройства поблизости, Bluetooth), а также убедитесь, что **GPS включен в шторке телефона**. Без этого радиомодули Android откажутся работать.

## 👨‍💻 Для разработчиков

Проект написан на **Flutter (Dart)** и вайбкоде и использует CI/CD пайплайны GitHub Actions для автоматической сборки без необходимости локальной установки тяжелого Android SDK.

### Структура проекта
- `lib/screens/` — Пользовательский интерфейс (Радар, Профиль, Чат).
- `lib/logic/app_state.dart` — Локальное хранилище данных и стейт-менеджер.
- `lib/logic/mesh_service.dart` — Сердце приложения: обертка над Google Nearby Connections API для управления P2P-сетью.
- `.github/workflows/build.yml` — Сценарий облачной компиляции и внедрения Android Permissions.

### Сборка из исходников
Если вы хотите собрать проект локально на своем ПК:
```bash
# 1. Установите Flutter SDK
# 2. Склонируйте репозиторий
git clone https://github.com/ВАШ-НИК/Bluessager.git
cd Bluessager

# 3. Скачайте зависимости
flutter pub get

# 4. Запустите на подключенном устройстве
flutter run --release
```
